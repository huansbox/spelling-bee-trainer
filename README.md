# Spelling Bee 訓練器

幫助小朋友準備 Spelling Bee 比賽的練習工具。

**Live Demo**: https://huansbox.github.io/spelling-bee-trainer/

## 功能特色

- **練習模式** - 單字卡瀏覽，點擊播放發音
- **預演模式** - 模擬比賽流程，家長出題、小孩口頭拼字
- **錯題集訓營** - 自動收集弱點單字，集中特訓
- **練習日誌** - 追蹤每日練習量與單字排行榜
- **進度同步** - 匯出/匯入代碼，支援跨裝置

## 技術

純前端單頁應用（HTML + JavaScript + TailwindCSS），無需後端。
